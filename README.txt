
Demon Slayer - Advanced Full Pack (3D + Menu + Combo + Stamina + Block)
======================================================================

İçerik:
- assets/models/player.obj & enemy.obj (placeholder 3D modeller)
- swing.wav, hit.wav, block.wav, special.wav, ambient_loop.wav, menu_music.wav
- ursina_advanced.py -> gelişmiş arena + combat sistemi
- README.txt

Kurulum:
1) Python 3.10+
2) pip install ursina
3) python ursina_advanced.py

Kontroller:
- WASD: Hareket
- Space / Sol Tık: Saldırı
- Shift: Blok
- Q: Dodge / hızlı hareket
- Esc: Çıkış

Notlar:
- Combo sistemi: her 3 normal saldırıda özel ses çalıyor
- Stamina tükenirse saldırı yapamaz
- Blok: gelen hasarı %50 azaltır ve stamina bir miktar geri kazanır
- Menü müziği ve ortam müziği ile basit atmosfer
- Model ve sesleri değiştirerek oyunu istediğin şekilde genişletebilirsin
